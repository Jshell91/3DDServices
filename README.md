# Servicio Web con Node.js y Express

Este proyecto es un ejemplo básico de un servicio web usando Node.js y Express.

## Cómo ejecutar

1. Instala las dependencias (si no lo has hecho):
   ```powershell
   npm install
   ```
2. Inicia el servidor:
   ```powershell
   npm start
   ```
3. Abre tu navegador y visita: [http://localhost:3000](http://localhost:3000)

Deberías ver el mensaje: `¡Hola, mundo! El servicio web está funcionando.`

## Estructura
- `index.js`: Código principal del servidor.
- `package.json`: Configuración del proyecto y dependencias.
